Description

WebSocket Server

This module maintains a WebSocket server which provides the backend services for Map display and Trainer tool.

websocket_handle(Data, Req, State) -> any()

Main server handler

Handles events from the frontent clients: Map and Trainer tool. Events and data are packed into JSON by the jsx library.

websocket_info/3

websocket_info(Info, Req, State) -> any()

Handles position data from the Analyzer

Called when the Analyzer sends location data on a device. The positional data is packed into JSON and sent to the Map

websocket_init/3

websocket_init(TransportName, Req, Opts) -> any()

Initializes a WebSocket process

Initializes a WebSocket process and registers itself as a subscriber to gproc's pubsub service, to receive broadcasts of data from the Analyzer.

websocket_terminate/3

websocket_terminate(Reason, Req, State) -> any()

Called when the WebSocket connection is ended

Unsubscribes itself from the publish-subscribe service provided by gproc when the WebSocket session ends.
